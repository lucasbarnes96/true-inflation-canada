2026-03-08 · v2.0.0

Lean Macro Refactor

  • Complete Backend Rewrite: Deleted the entirety of the legacy FastAPI/Apify data pipeline to focus purely on vast macro-economic indicators.
  • Auto-Pilot Architecture: Introduced a fully autonomous GitHub Actions pipeline that generates a static flat JSON file (`chart_data.json`) nightly, radically simplifying deployment and hosting.
  • Interactive Deep Charting: Added sophisticated dual-line charting capabilities allowing users to plot any asset against any inflation adjuster (M1+, M3, CPI) dynamically.
  • Dynamic Text Content: The application now computes accurate YoY percentages and metrics entirely client-side based on the latest data inputs.
  • Optimized Link Sharing: Injected full suites of Open Graph and Twitter Card tags to all routing endpoints to guarantee perfect unfurling across social media platforms.
Experimental open-source project. Not official StatCan CPI.
