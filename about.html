Transparent Inflation Tracking on Public Data

True Inflation Canada tracks the true value of the Canadian Dollar and various assets by adjusting them against macro-economic parameters like Bank of Canada's M3 money supply and the official Consumer Price Index. The project is designed to be lean, robust, and completely transparent.

Architecture

Fully Open Source

The entire ingestion pipeline and rendering logic is publicly available on GitHub. You can audit every data source and formula we use to calculate the charts.

No Hidden Backends

We don't use proprietary databases or black-box APIs. All data is fetched directly from public sources (Bank of Canada, StatCan, Yahoo Finance), compiled into a flat JSON payload by a transparent Github Action, and rendered in your browser.

Data Sources

Monetary Adjusters

  • M1+, M3: Sourced via the official Bank of Canada Valet API.
  • CPI: Sourced via the official Bank of Canada Valet API (All-items).

Assets & Equities

  • Indices (TSX, S&P 500), Commodities, Crypto: Sourced via Yahoo Finance historical data. Non-CAD assets are mathematically converted to CAD at run-time using current exchange rates.
  • Housing & Labour: Sourced via Statistics Canada raw CSV dumps.
